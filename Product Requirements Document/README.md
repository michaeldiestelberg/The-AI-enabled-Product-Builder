# Product Requirements Document Skill

A Claude skill that guides users through creating comprehensive product requirements documents (PRDs) from initial product ideas through an interactive, workshop-style conversation. Designed for product builders who need structured requirements documentation with user stories, business rationale, and acceptance criteria through collaborative discovery and iterative feedback.

- Claude
    - Review the SKILL.md file and modify it as needed
    - Add SKILL.md to a zip archive and upload the skill to Claude (for web and app use)
- ChatGPT
    - Copy the contents of SKILL.md and add it as an instruction to your custom GPT
    - Alternatively, use my shared GPT: [Product Requirements Document GPT](https://chatgpt.com/g/g-69312d8da5cc8191b7b73637238bbe4a-product-requirements-document-prd)
- Gemini
    - Copy the contents of SKILL.md and add it as an instruction to your custom Gem
    - Alternatively, use my shared Gem: [Product Requirements Document Gem](https://gemini.google.com/gem/1ZVUNMGsvDpafQUPeLBDjefblV9VXvPcn?usp=sharing)
