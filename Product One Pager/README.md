# Product One-Pager Creation Skill

A Claude skill that helps product managers create focused, concise product one-pagers through guided conversation. Designed for product builders who need to articulate the why and what they're building before diving into development. Perfect for product opportunities, PRD precursors, or vibe coding kickoffs.

## Configuration

- Review the SKILL.md file and modify it as needed
- Upload the skill to Claude (for web and app use)
