# Today's Focus Tasks Skill

A Claude skill that automates the transfer of focus tasks from your daily "@Today" note in Notion to Todoist. Designed for product builders who want to create Todoist tasks from their daily planning note in Notion without manual copying.

## Configuration

- Review the SKILL.md file and modify it as needed
- Ensure the Notion note title and Todoist project name in SKILL.md match your setup (defaults are "@Today" and "Mapp")
- Upload the skill to Claude (for web and app use)
