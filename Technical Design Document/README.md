# Technical Design Document Creation Skill

A Claude skill that guides product builders through creating comprehensive technical design documents via an interactive workshop process. Takes a product opportunity or brief and collaboratively develops a detailed technical spec through discovery, scoping, technology choices, and architecture decisions. Designed for teams that need to move from "what to build" to "how to build it" with clarity and structure.

## Configuration

- Review the SKILL.md file and modify it as needed
- Upload the skill to Claude (for web and app use)
